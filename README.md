# Signature Blue Star

Desktop app built with Electron to:

- Open and preview PDF files
- Select an area on a page with the mouse
- Place an electronic signature block on the selected area
- Save the signed PDF as a new file

## Run

```powershell
npm install
npm start
```

## Notes

- This app currently adds a visible electronic signature block (text overlay) to the PDF.
- It does not perform certificate-based PKI digital signing with USB token / HSM.
